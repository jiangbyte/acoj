# 功能特性

Hei Admin Vue 提供了一套完整的后台管理系统前端通用能力。

## 核心功能清单

- [**认证体系**](auth)：SM2 国密加密登录、JWT 令牌管理、双端（B/C）认证体系
- [**权限管理**](permission)：RBAC 权限模型、路由级权限控制、按钮级权限过滤
- [**主题与布局**](theme-layout)：明亮/暗黑双主题、主色定制、多布局模式、响应式适配
- [**组件体系**](components)：CRUD 表格、树形表格、搜索面板、抽屉表单等高级业务组件
- [**HTTP 请求封装**](http)：Alova 请求库封装、统一错误处理、令牌刷新

## 功能矩阵

| 功能 | 说明 |
|------|------|
| SM2 加密登录 | 国密 SM2 C1C3C2 模式加密传输密码 |
| SM 哈希 | 全局字典管理，支持标签值和颜色映射 |
| JWT 令牌管理 | Token 自动携带、过期自动跳转登录 |
| RBAC 权限 | 用户→角色→权限 + 用户直授权限 |
| 路由级权限 | 根据后端菜单配置动态生成路由 |
| 按钮级权限 | 根据权限码过滤页面中的按钮 |
| 动态路由 | 从后端菜单配置动态生成 Vue Router |
| 主题切换 | 浅色/深色主题，一键切换 |
| 主色定制 | Ant Design 主题算法，动态主色切换 |
| 多布局 | 垂直布局模式（支持扩展） |
| 响应式 | 桌面端和移动端自适应，移动端抽屉导航 |
| 标签页 | 多标签页导航，支持缓存/刷新/关闭 |
| 字典系统 | 全局字典缓存，模板中直接使用 |
| 文件上传 | 支持文件预览、下载、图片裁剪上传 |
| 操作日志 | 操作日志和访问日志的查询展示 |
| 在线监控 | B 端和 C 端的在线用户会话管理 |
