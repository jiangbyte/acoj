# 功能特性

Hei Gin 提供了一套完整的后台管理系统通用能力，覆盖认证、权限、日志、存储等核心领域。

## 核心功能清单

- [**双端认证**](auth)：B 端（管理端）和 C 端（客户端）完全隔离的 JWT 认证体系
- [**权限管理**](permission)：RBAC 权限模型、权限自动发现、通配符匹配、行级数据权限
- [**中间件体系**](middleware)：完整的中间件链，涵盖链路追踪、认证、权限、日志、防重复等
- [**文件存储**](storage)：统一的文件存储接口，支持本地/MinIO/S3 三种后端
- [**操作日志**](log)：全量操作日志录制，支持请求/响应录制和 SM3 防篡改签名
