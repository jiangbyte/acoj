# 介绍

## Hei Admin Vue 是什么？

**Hei Admin Vue** 是 HEI 快速开发框架的 Vue3 前端管理后台解决方案，与后端框架（Hei Gin、Hei Boot 等）配套使用，提供完整的前后端分离架构。

## 设计理念

- **约定优于配置** — 统一的目录结构和编码规范，减少团队决策成本
- **开箱即用** — 内置完整的权限管理、路由守卫、HTTP 封装等通用能力
- **可扩展性** — 模块化的页面组织和组件设计，方便业务功能扩展
- **现代化技术栈** — 采用最新的前端技术，保持技术前瞻性

## 适用场景

- 企业级后台管理系统
- SaaS 平台管理端
- 内容管理后台
- 数据监控和仪表盘
- 任何需要快速搭建后台界面的项目

## 核心能力

| 能力 | 说明 |
|------|------|
| 认证授权 | SM2 加密登录、JWT 令牌管理、双端（B/C）认证体系 |
| 权限管理 | RBAC 权限模型、路由级权限控制、按钮级权限过滤 |
| 动态路由 | 根据后端菜单配置动态生成前端路由 |
| 主题系统 | 明亮/暗黑主题切换、主色定制、布局模式选择 |
| 响应式适配 | 桌面端和移动端自适应布局 |
| HTTP 封装 | Alova 请求库封装，统一的错误处理和令牌刷新 |
| 字典系统 | 全局字典管理，支持标签和颜色映射 |
| 页面组件 | CRUD 表格、树形表格、搜索面板、抽屉表单等高级组件 |
